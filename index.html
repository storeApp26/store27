🔥 إعداد Firebase

هذا الإعداد مرة واحدة فقط — بعده المتجر يعمل بشكل حقيقي على الإنترنت لجميع الزبائن.

1

أنشئ مشروع Firebase مجاني

اذهب إلى console.firebase.google.com → أنشئ مشروعاً جديداً → اختر أي اسم

2

فعّل Firestore Database

من القائمة الجانبية → Firestore Database → Create Database → اختر Test mode → Next → Enable

3

فعّل Storage (للصور)

من القائمة الجانبية → Storage → Get Started → Next → Done

4

احصل على إعدادات المشروع

⚙️ Project Settings → General → مرر للأسفل → Your apps → اضغط </> → Register app → انسخ الـ Config
